# Bun venit la Riverbed Activity Tracker Project! 👋

> [!NOTE]
> Acest seif (Vault) Obsidian a fost conceput special pentru tine ca **Software Engineering Intern**. Scopul său este să te ghideze rapid prin codul, structura și sarcinile pe care le ai de îndeplinit.

Proiectul pe care lucrezi este un **API de urmărire a activității utilizatorilor** (Activity Tracker), dezvoltat în Python folosind FastAPI. Deși codul este compact, el include elemente esențiale pe care le vei întâlni zilnic: stocare in-memory, modele de validare a datelor (Pydantic), endpoint-uri RESTful și teste automate.
